# Excel File Examples

Sample files for testing the candidate registration feature.

## File Format

| Column       | Type    | Values               |
| ------------ | ------- | -------------------- |
| seniority    | string  | junior, senior       |
| years        | number  | 0-50                 |
| availability | boolean | true, false, yes, no |

## Files

- `candidate-senior-available.xlsx` - Senior, 5 years, available
- `candidate-junior-available.xlsx` - Junior, 2 years, available
- `candidate-senior-unavailable.xlsx` - Senior, 10 years, unavailable
